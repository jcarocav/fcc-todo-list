# 📝 My To-Do List - freeCodeCamp Project

Este es un proyecto de lista de tareas creado como parte del curso de **freeCodeCamp**. El objetivo es practicar la estructura de listas anidadas en HTML y el uso de pseudo-clases en CSS.

## 🚀 Tecnologías utilizadas
*   **HTML5:** Estructura semántica y formularios.
*   **CSS3:** Diseño responsivo, tarjetas (cards) y efectos de estado (:hover, :focus, etc).

## 🎨 Características
*   Diseño limpio con tarjetas verdes claras.
*   Checkbox funcional para marcar tareas.
*   Enlaces integrados a recursos externos.
*   Totalmente adaptable a diferentes tamaños de pantalla.
